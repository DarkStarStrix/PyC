# Security Policy

## Supported Versions

PyC is pre-1.0. Security fixes are provided on the default branch and the latest release tag.

| Version | Supported |
|---|---|
| Latest release | Yes |
| `main` | Yes |
| Older tags | No |

## Reporting a Vulnerability

Please do not open public GitHub issues for unpatched security vulnerabilities.

Report privately by email:

- `allanw.mk@gmail.com`

Include:

1. A clear description of the issue.
2. Steps to reproduce.
3. Affected commit/tag and environment.
4. Any proof-of-concept or logs.

## Response Expectations

1. Initial triage response target: within 3 business days.
2. Confirmed issues will be tracked privately until a fix is ready.
3. After release, a public advisory/changelog note will be published.
