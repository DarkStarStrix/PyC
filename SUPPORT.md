# Support

## Where to Get Help

- Usage questions: open a GitHub Discussion (if enabled) or issue with the `question` label.
- Bug reports: open a GitHub issue and include reproduction details.
- Security issues: follow `SECURITY.md` and report privately.

## What to Include

1. OS and compiler/toolchain version.
2. CMake version.
3. Exact commands run.
4. Logs/errors and expected behavior.
5. Minimal reproducible input if applicable.

## Scope

Support is best-effort for:

- latest release,
- current `main` branch,
- Linux/macOS/Windows CI-supported paths.
